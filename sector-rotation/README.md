# 台股板塊輪動追蹤平台

一個實時監測台灣股市板塊資金流動的追蹤平台，幫助投資人觀察市場資金動向。

## 功能特性

### 核心功能
- 🫧 **泡泡圖可視化** - 直觀展示板塊表現
  - X軸：近5日表現
  - Y軸：近20日累計表現  
  - 圓大小：代表收益幅度

- 📊 **排行榜視圖** - 詳細的數據表格展示
  - 支持多種排序方式
  - 快速查看板塊詳情

- 💰 **資金狀態分類**
  - 主力：資金加速流入 🟢
  - 輪動：資金流入但放緩 🔵
  - 觀望：資金沉寂 🟡
  - 退潮：資金流出 🔴

### 互動功能
- ✨ 即時數據刷新（每5分鐘自動更新）
- 🔍 資金狀態篩選
- 🔎 自由縮放和拖曳
- 📱 完全響應式設計
- 📋 詳細信息面板

## 技術棧

### 前端
- HTML5 / CSS3 / Vanilla JavaScript
- D3.js - 數據可視化
- 響應式設計

### 後端
- Vercel Serverless Functions (Node.js)
- 數據爬蟲（從公開數據源獲取）

### 部署
- Vercel - 完整託管方案
- 無需自建伺服器

## 本地開發

### 安裝依賴
```bash
npm install
```

### 本地開發服務
```bash
npm run dev
```

訪問 `http://localhost:3000`

### 構建
```bash
npm run build
```

### 部署到 Vercel
```bash
npm run deploy
```

## 數據源

目前使用模擬數據進行演示。實際部署可集成以下數據源：
- **台灣證券交易所 (TWSE)**：公開市場數據
- **投信投顧公會**：法人資金流向數據
- **其他金融數據 API**
