## 🎓 教學內容詳解

### 第1章：基礎概念 (5-10分鐘)
**目標**: 掌握時間單位轉換

**內容**:
- 1天 = 24小時
- 1小時 = 60分鐘
- 1分鐘 = 60秒
- 交互式換算器（輸入天數自動計算）

**交互**:
```html
<input id="days" placeholder="輸入天數">
<!-- 自動顯示 日 = X小時 = Y分鐘 = Z秒 -->
```

### 第2章：跨日正推 (10-15分鐘)
**目標**: 已知起始時間和經過時間，求結束時間

**經典例題**:
```
周一下午3點開始，經過30小時後是？
30小時 = 1天 + 6小時
下午3點 + 6小時 = 下午9點（第一天）
下午9點 + 1天 = 周二下午9點（答案）
```

**解題步驟**:
1. 把小時轉換為天+小時
2. 先加小時
3. 再加天數

### 第3章：跨日反推 (10-15分鐘)
**目標**: 已知結束時間和經過時間，求起始時間

**經典例題**:
```
今天早上8點，已經是課程啟動後的第30小時
課程開始時間是？

30小時 = 1天 + 6小時
早上8點 - 6小時 = 凌晨2點（前一天）
凌晨2點 - 1天 = 前天凌晨2點（答案）
```

**反推技巧**:
- 把小時轉換為天+小時
- 先減小時
- 再減天數
- 注意借位（小時不足時借1天）

### 第4章：交互練習 (10分鐘)
3個可拖拽時間軸例題，用戶可以手動推進/回退時間

### 第5章：本章小測 (5分鐘)
5道快速評估題，通過後才能進入正式題庫

## 📊 題庫配置

### 基礎題 (15道)
簡單單位轉換
- 1天 = 多少小時？
- X小時 = 多少天？
- 複合單位（1天2小時 = 多少小時？）

**難度**: ⭐ 簡單

### 進階題 (20道)
跨日正推計算
- 周一早上8點，經過36小時後是周幾幾點？
- 涉及 24 小時交界線
- 需要分步驟計算

**難度**: ⭐⭐ 中等

**類型**: 步驟式答題
- 第1步: 分解小時
- 第2步: 加上小時部分
- 第3步: 加上天數部分

### 挑戰題 (15道)
跨日反推計算
- 周四晚上10點，已經是某活動開始後的第56小時。何時開始？
- 需要反向思考
- 可能涉及借位計算

**難度**: ⭐⭐⭐ 困難

**類型**: 步驟式答題（反向）

## 🚀 使用說明

### 開始使用

1. **進入模塊**
   - 在首頁點擊「🕐 時間換算專項」按鈕
   - 自動加載教學第一章

2. **學習流程**
   ```
   教學第1-5章 → 本章小測（5題）→ 基礎題庫（15題）
   → 進階題庫（20題） → 挑戰題庫（15題）
   ```

3. **答題流程**
   - 閱讀題目
   - 選擇或輸入答案
   - 提交驗證
   - 查看詳細解析

4. **導航**
   - 返回首頁: 點擊「← 返回首頁」
   - 快捷鍵: 按 Esc 返回首頁

### 特殊功能

**提示系統**:
- 進階題和挑戰題有 💡 查看提示 按鈕
- 幫助用戶理解解題邏輯

**重試機制**:
- 答錯可以重新試答
- 不會立即扣分或結束

**進度追蹤**:
- 實時顯示 「第 X / Y 題」
- 進度條視覺化
